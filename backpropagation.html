Visualização do Backpropagation

Entenda como o algoritmo de retropropagação atualiza os pesos da rede neural

Teoria do Backpropagation

O backpropagation (ou algoritmo de retropropagação) é fundamental para o treinamento de redes neurais. Ele consiste em três fases principais:

  1. Fase Forward (Propagação): Os dados de entrada passam pela rede, da camada de entrada até a saída, gerando uma predição.
  2. Cálculo do Erro: A saída da rede é comparada com o valor esperado, e um erro é calculado.
  3. Fase Backward (Retropropagação): O erro é propagado para trás através da rede, calculando como cada peso contribuiu para o erro.
  4. Atualização dos Pesos: Os pesos são ajustados na direção que reduz o erro, usando a regra do gradiente descendente.

A intuição por trás do algoritmo é simples: se um peso contribui para um erro maior, ele deve ser ajustado na direção oposta. O algoritmo calcula o gradiente do erro em relação a cada peso e usa essa informação para ajustar os pesos de forma a minimizar o erro.

Na ferramenta acima, você pode visualizar este processo passo a passo, observando como os pesos mudam e como o erro diminui ao longo do tempo. As cores e espessuras das conexões representam os valores dos pesos e gradientes.
